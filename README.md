# 📊 Student Data Analysis Project

## Overview
This project analyzes a dataset of 9000 students using Python and Pandas.

##  Tasks Performed
- Created Total Marks column
- Found Top 10 students
- Calculated subject-wise averages
- Analyzed grade distribution
- Identified subject toppers

##  Tools Used
- Python
- Pandas

## Key Insight
Most students scored Grade D

##  How to Run
1. Install pandas
2. Run: python student_analysis.py
